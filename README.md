# SeeArt

SeeArt tracks current and upcoming exhibitions for some museums and art galleries in **NYC, Boston, Philadelphia, London, Munich, and Moscow** (this list is to be expanded) to get rid of the long manual website hopping when you want to go see an exhibition but do not know what is on.

We refresh the data weekly on Mondays (8-9am EST).


## Add Venues, Tabs, Or Cities

Edit:

```text
config/venues.json
```

Each venue belongs to a city and tab. Later cities can be added by adding more venue entries with a new `city` value.
