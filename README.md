# SeeArt

SeeArt tracks current and upcoming exhibitions for museums and art galleries in **NYC, Boston, Philadelphia, London, Munich, and Moscow**, getting rid of the long manual website-hopping when you want to go see an exhibition but do not know what is on.

For NYC, SeeArt also tracks public one-off museum events and tours, excluding member-only programs, classes, workshops, camps, and courses.

We refresh the data daily around 9am New York time.

New venues and cities can be added by editing ```config/venues.json```.
